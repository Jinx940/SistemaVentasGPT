# Frontend de SistemaVentasGPT

Aplicacion React/Vite preparada para Vercel.

## Desarrollo local

```powershell
npm install
npm run dev
```

Configura `VITE_API_URL` usando `.env.example` como referencia.

## Produccion

En Vercel usa `frontend` como Root Directory y define:

```text
VITE_API_URL=https://TU-BACKEND.vercel.app
```

Consulta [`../DEPLOY.md`](../DEPLOY.md) para el despliegue completo.
